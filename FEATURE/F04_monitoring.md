# F04 — 현황 모니터링

## 개요

주문 처리 현황과 시료 재고 상태를 한눈에 파악할 수 있는 조회 기능.

---

## 요구사항

### F04-1 주문 현황 요약

- 현재 주문 수를 상태별로 표시한다.
  - 표시 대상 상태: 접수(RESERVED) / 확정(CONFIRMED) / 생산중(PRODUCING) / 출고(RELEASE)
  - 거절(REJECTED) 주문은 현황에서 제외한다.

### F04-2 시료별 재고 상태

- 등록된 모든 시료의 현재 재고 수량과 상태 레이블을 표시한다.
- 재고 상태 레이블 기준:
  - **여유**: 재고가 해당 시료의 전체 미처리 주문 수량보다 많은 경우
  - **부족**: 재고가 0보다 크지만 전체 미처리 주문 수량 이하인 경우
  - **고갈**: 재고가 0인 경우

### F04-3 화면 갱신

- 모니터링 화면에서 명령을 입력할 때마다 최신 데이터로 갱신된다.

---

## 제약

- 모니터링 화면에서는 데이터를 수정할 수 없다. 조회 전용이다.
