# F03 — 주문 승인·거절

## 개요

접수된 주문을 담당자가 검토하여 승인하거나 거절하는 기능.  
승인 시 재고 상황에 따라 결과가 달라진다.

---

## 요구사항

### F03-1 주문 승인

- 담당자는 접수(RESERVED) 상태인 주문을 승인할 수 있다.
- 승인 시 시스템이 재고를 자동으로 확인한다.

**재고가 충분한 경우:**
- 주문 상태가 **확정(CONFIRMED)** 으로 변경된다.
- 담당자에게 "주문이 확정되었습니다" 메시지를 표시한다.

**재고가 부족한 경우:**
- 주문 상태가 **생산중(PRODUCING)** 으로 변경된다.
- 부족한 수량만큼의 생산 작업이 생산라인 대기 큐에 자동 등록된다.
- 담당자에게 "재고 부족으로 생산이 배정되었습니다" 메시지를 표시한다.

### F03-2 주문 거절

- 담당자는 접수(RESERVED) 상태인 주문을 거절할 수 있다.
- 거절 시 주문 상태가 즉시 **거절(REJECTED)** 로 변경된다.
- 거절된 주문은 이후 어떠한 처리도 진행되지 않는다.

### F03-3 처리 대상 목록

- 승인·거절 화면에서는 접수(RESERVED) 상태 주문만 표시된다.

---

## 제약

- 이미 승인·거절된 주문은 다시 처리할 수 없다.
- 거절 취소(복구) 기능은 제공하지 않는다.
