# F02 — 시료주문 접수

## 개요

고객으로부터 시료 주문을 받아 시스템에 등록하는 기능.

---

## 요구사항

### F02-1 주문 접수

- 담당자는 고객의 시료 주문을 시스템에 입력할 수 있다.
- 입력 항목: 고객명, 시료 ID (또는 이름), 주문 수량.
- 접수 즉시 주문 ID가 자동 부여되고, 상태는 **접수(RESERVED)** 로 설정된다.
- 접수 완료 후 주문 ID와 현재 상태를 화면에 표시한다.

### F02-2 주문 유효성 검사

- 시스템에 등록되지 않은 시료는 주문할 수 없다.
  - 없는 시료 ID 입력 시: "등록되지 않은 시료입니다" 메시지 표시 후 재입력.
- 주문 수량은 1 이상이어야 한다.
- 고객명은 비어 있을 수 없다.

### F02-3 주문 목록 조회

- 접수된 전체 주문 목록을 볼 수 있다.
- 목록에는 주문 ID, 고객명, 시료명, 수량, 현재 상태, 접수 일시가 표시된다.

---

## 제약

- 주문 접수 단계에서는 재고 확인이 일어나지 않는다. 재고 판단은 승인 시점에 이루어진다.
- 동일 고객이 같은 시료를 여러 번 주문할 수 있다 (중복 허용).
