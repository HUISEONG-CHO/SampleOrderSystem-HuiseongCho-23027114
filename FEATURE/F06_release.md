# F06 — 출고처리

## 개요

확정된 주문을 최종 출고 처리하여 고객에게 인도하는 기능.

---

## 요구사항

### F06-1 출고 대상 목록 조회

- 확정(CONFIRMED) 상태인 주문 목록을 표시한다.
- 표시 항목: 주문 ID, 고객명, 시료명, 주문 수량, 확정 일시.

### F06-2 출고 실행

- 담당자는 목록에서 특정 주문을 선택해 출고를 진행할 수 있다.
- 출고 실행 시:
  - 해당 시료의 재고에서 주문 수량만큼 차감된다.
  - 주문 상태가 **출고(RELEASE)** 로 변경된다.
- 출고 완료 후 처리 결과를 화면에 표시한다.

---

## 제약

- 확정(CONFIRMED) 상태가 아닌 주문은 출고할 수 없다.
- 출고 취소(반품) 기능은 제공하지 않는다.
- 재고가 주문 수량보다 적은 상태에서는 출고를 진행할 수 없다.
  - 정상적인 흐름(승인 → 생산 → 확정)에서는 이 상황이 발생하지 않아야 한다.
